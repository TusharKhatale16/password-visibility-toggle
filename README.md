# Password Visibility Toggle

A simple, lightweight UI component to toggle password visibility (show/hide) in input fields. Built with HTML, CSS, and JavaScript.
link ["https://password-toggler-new.netlify.app/"]

## Features

-  Toggle password visibility with a single click
-  Clean and responsive UI design
-  Lightweight and fast
-  Mobile-friendly
-  Easy to integrate into forms

## Demo

Simply open `index.html` in your browser to see the password visibility toggle in action.

## Installation

1. Clone or download this repository:
   ```bash
   git clone https://github.com/yourusername/password-visibility-toggle.git
   ```

2. Navigate to the project directory:
   ```bash
   cd password-visibility-toggle
   ```

3. Open `index.html` in your web browser.

## How It Works

1. The toggle button allows users to show or hide the password in the input field
2. Click the eye icon to toggle between password and text visibility
3. The password field updates in real-time

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling and design
- **JavaScript (Vanilla)** - Interactivity

## Usage

Include the HTML input field with the toggle button, and the JavaScript will handle the show/hide functionality automatically.
